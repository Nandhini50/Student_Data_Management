# Student Data Management System

## Project Description

This project is a Student Data Management System developed using HTML, CSS, JavaScript, Node.js, and Express.js.

The main purpose of this project is to manage student details and marks in a simple and organized way. The system has two types of users:

- Admin
- Student

The Admin can add, update, and delete student records and manage marks. Students can log in and view their personal details and marks.

---

## Features

### Login System
- User login
- User registration
- Role-based access (Admin/Student)

### Admin Features
- Add student details
- Update student information
- Delete student records
- Add and update marks
- View all student data

### Student Features
- Login securely
- View personal details
- View marks and grades

### Marks Management
- Internal marks
- External marks
- Total calculation
- Average calculation
- Grade generation
